## 🎯 What is QRGanic?

QRGanic is a **multi-role agricultural e-commerce platform** that connects:
- 👥 **Users** - Browse and purchase certified products
- 🌾 **Farmers** - Sell certified agricultural products
- 🏢 **FPOs** - Verify and certify farmers and products
- 👨‍💼 **Admins** - Manage the entire platform

## ✨ Key Features

✅ **Multi-Portal System** - Separate dashboards for each role  
✅ **Product Certification** - Only FPO-verified products  
✅ **QR Code Generation** - For every purchase  
✅ **Email Notifications** - Automated alerts for all actions  
✅ **Location-Based Routing** - FPOs assigned by location  
✅ **Dark Green Theme** - Beautiful, modern UI  

## 🛠️ Technology Stack

- **Backend:** Node.js + Express.js + MongoDB
- **Frontend:** React + Vite
- **Authentication:** JWT
- **Email:** Nodemailer
- **QR Codes:** QRCode library

## ⚡ Quick Commands

```bash
# Install all dependencies
npm run install-all

# Start backend (Terminal 1)
npm run dev-backend

# Start frontend (Terminal 2)
npm run dev-frontend
```

## 🔐 Default Test Accounts

After registration, you can create:
- **Admin:** Full system access
- **FPO:** Manage farmers and products
- **Farmer:** Upload and sell products
- **User:** Browse and purchase products

## 📧 Email Setup

For email notifications to work:
1. Use a Gmail account
2. Enable 2-Step Verification
3. Generate App Password
4. Add to `backend/.env`

## 🆘 Need Help?
